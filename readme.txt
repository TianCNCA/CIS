To run this app, execute Complie.bat, then Run.bat
To run all the unit tests, execute RunUnitTests.bat
To run all the integration tests, execute RunIntegrationTests.bat

If you encountered any any error message related to DB(very unlikely, we've made a lot of effort on it), please execute database/WipeDB.bat

The documents are all in the documents folder.	
Our repo on github is at: https://github.com/NameCT/CIS
